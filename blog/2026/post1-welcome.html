    I guess I'll write an introduction to the site in general here.

    So welcome to my personal corner of the web. The short story about this site is that it was supposed to just host some links, but got a bit out of hand. (Although it might just end up hosting links either way.) This became quite a learning project into CSS, HTML and JavaScript. I also ran into Neocities, which has been a good source of inspiration. Creating a personal social media page from scratch is certainly more fun than playing completely by the rules of massive megacorporations. And the best part is that anyone can make one since the main languages to make static websites, CSS and HTML, are easy to learn.

    As you might be able to tell by the site address, this site is not hosted on Neocities, but on GitHub (owned by a megacorporation btw :D). The source code for this site is completely public and can be found on my GitHub. More documentation about the site and other stuff can be found in the Documentation (not made yet).

    I personally use Firefox based browsers and the website has them in mind. It should work on Chromium based ones as well, although there may be visual differences. It's also worth noting that site usability with larger screens comes before smaller ones like mobile devices.

    The default theme is dark but if you'd prefer a light theme, it can be switched on from the menu, accessed by clicking the three lines in the top-right corner. And feel free to leave a message in the Guestbook , hosted on Atabook.
